# CyberSecurity Lab

A personal cybersecurity portfolio documenting hands-on work across threat detection, malware analysis, and capture-the-flag competitions. Each section reflects practical skills developed through building and operating a home lab environment.

---

## Contents

### 🛡️ SIEM Lab
Documentation of a self-hosted Security Information and Event Management environment, including architecture decisions, detection rule development, log ingestion configuration, and alert tuning.

### 🔬 Malware Analysis
Static and dynamic analysis write-ups of malware samples, covering techniques such as PE inspection, behavioral analysis, network indicator extraction, and sandbox detonation.

### 🚩 CTF Write-ups
Detailed solutions to Capture the Flag challenges across network forensics, memory analysis, and threat hunting. Each write-up documents the methodology, tools used, and key findings.
